## Swift版本区别

### Swift 5
```

1. 稳定的ABI
2. 字符串支持 \() 与 #
3. 枚举值支持数组
4. @PropertyWapper
5. @dynamicCallable 
6. async 和 await
7. keyPath支持 identity修改自身
8. 结构体初始化修改
9. some关键字-编译透明
10. 申明与系统类相同的对象名称
11. SwiftUI
12. 单行表达式隐式返回
```

### SWift 4
```
1. Codeable协议
2. @dynamicMemberLoopup
3. keyPath
4. 条件一致性
5. CaseIterProtocol
6. API 
7. #warning 与 #error 编译
8. 下标
9. 类型组合 &
10. associatedType 增加where语句
```
